Live · v1.0

CAPTCHA Solver

A headless, async CAPTCHA solving service built with Camoufox. Supports Cloudflare Turnstile, reCAPTCHA v2, v2 Invisible, v2 Enterprise, v3, and v3 Enterprise.

Base URL http://127.0.0.1:{PORT}
Endpoints
GET Cloudflare Turnstile
/turnstile

Solves Cloudflare Turnstile challenges. Submits the task asynchronously and returns a task_id — poll /result to retrieve the token.

Parameters
NameDescriptionStatus
urlTarget page URLrequired
sitekeyTurnstile site keyrequired
actionTurnstile action valueoptional
cdataTurnstile custom dataoptional
Example
GET /turnstile?url=https://example.com&sitekey=0x4AAAAAAABBBBcc...
Responses
202Task accepted — returns task_id
400Missing required parameters
429Server at max capacity
Full URL 
http://127.0.0.1:5032/turnstile?url=https://example.com&sitekey=0x4AAAAAAABBBBccDDeE1122&action=login&cdata=myCustomData
202Response
application/json
{
  "task_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "status":  "accepted"
}
200Result (via /result)
application/json
{
  "status":       "success",
  "elapsed_time": 2.431,
  "value":        "0.xxxxxxxx-yyyyyyyy-zzzzzzzz..."
}

GET Get Result
/result

Poll this endpoint to retrieve the solving result using the task_id returned when a task was submitted. Tasks expire after 5 minutes.

Parameters
NameDescriptionStatus
idThe task ID from submissionrequired
Example
GET /result?id=a1b2c3d4-...
Responses
200Solved — token returned in value
202Still processing — poll again
404Invalid or expired task ID
408Task timed out (5 min limit)
Full URL
http://127.0.0.1:5032/result?id=a1b2c3d4-e5f6-7890-abcd-ef1234567890
200Success
application/json
{
  "status":       "success",
  "elapsed_time": 2.431,
  "value":        "0.xxxxxxxx-yyyyyyyy-zzzzzzzz..."
}
202Still Processing
application/json
{
  "status":     "process",
  "message":   "solving turnstile",
  "start_time": 1706000000.123
}
408Timeout
application/json
{
  "status":       "error",
  "elapsed_time": 300.012,
  "value":        "timeout",
  "message":      "Task timeout"
}
Requirements

Python

Python 3.9 or higher is required to run the server.

python ≥ 3.9

Dependencies

Install all required packages via pip before starting.

fastapi uvicorn camoufox loguru

System

Camoufox requires a browser binary. Run the download once before first use.

camoufox download

Network

Outbound internet access is required to reach Cloudflare & Google CAPTCHA servers.

proxy (optional)
